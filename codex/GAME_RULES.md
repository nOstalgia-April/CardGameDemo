# 9宫格规则草案

## 回合与费用
- 每回合只能翻一个面。
- 费用每回合 +1，默认起始为 1。
- 费用用于【移动 + 放怪】，共享同一费用池。
- 手牌在回合开始时一次性全部发放。

## 视野（战争迷雾）
- 第一回合：棋盘全可见（3x3）。玩家可选择任意空闲格子放置棋子。
- 之后：仅以下格子可见，其余不可见：
  - 怪物的上下左右格子
  - 棋子移动过的格子

## 怪物与 Boss
- 怪物回合结束时，从北到西依次攻击一轮（方向顺序：北 → 东 → 南 → 西）。
- 怪物攻击不相消。
- 每一关有一个 Boss。
- Boss 会移动：用曼哈顿距离找最近棋子，随机选择其一进行追击。
- 其他怪物不会移动。

## 攻击与效果
- 正常“攻击”：相消双方对应方向的数值，数值降为 0 后，下一次来自该方向的攻击会摧毁棋子。
- “优势”：攻击时自身对应方向不掉血。
- “冲锋”：
  - 放置时消耗一次翻牌次数。
  - 自动对四周发起一次“优势”攻击。
- “圣盾”：
  - 放置时消耗一次翻牌次数。
  - 获得一次放置时相邻怪物方向的格挡效果，可免疫下一次来自该方向的攻击。
  - （可能的设计：圣盾与冲锋在手牌中确定效果方向，每回合切换）

## 翻牌
- 每回合仅能翻牌一次。
- 翻牌效果：恢复相邻棋子满额生命（包括敌方）。
